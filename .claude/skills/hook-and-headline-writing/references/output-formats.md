## Output File Naming Convention

**Pattern:** `[Date/Topic]_[Type]_[Status].md`

**Examples:**
- `2025-10-28_Subject_Lines_Final.md`
- `Productized_Services_Thread_Hooks_Draft.md`
- `Landing_Page_Headlines_V2.md`

**Status indicators:**
- `_Draft` - First round, needs review
- `_Revised` - After feedback incorporated
- `_Final` - Approved and ready to use
- `_Testing` - Currently in A/B test
- `_Winner` - Test concluded, this won
